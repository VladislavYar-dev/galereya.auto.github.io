# Каталог автомобилей

## Структура папок

```
catalog/
  bmw-5-2021/
    1.jpg   ← главное фото (показывается первым)
    2.jpg
    3.jpg
  mercedes-e-2022/
    1.jpg
    2.jpg
  ...
```

## Как добавить новый автомобиль

1. Создайте папку: `catalog/<id>/`  
   Имя папки — уникальный slug, например: `kia-sportage-2023`

2. Положите фотографии: `1.jpg`, `2.jpg`, `3.jpg` ...  
   Рекомендуемый размер: **1200×750 px**, формат JPG или WebP.

3. Откройте файл `catalog.js` и добавьте объект в массив `CATALOG`:

```javascript
{
  id: "kia-sportage-2023",
  name: "Kia Sportage",
  year: 2023,
  origin: "из Кореи",
  engine: "1.6 · 180 л.с.",
  mileage: "18 000 км",
  transmission: "Автомат",
  price: "69 900 BYN",
  priceNote: "или в кредит от 920 BYN/мес.",
  badge: "Новое",         // null — без ярлыка
  badgeColor: "#e8c96a",  // null — золотой по умолчанию
  description: "Полный привод, панорамная крыша, подогрев сидений...",
  photos: [
    "catalog/kia-sportage-2023/1.jpg",
    "catalog/kia-sportage-2023/2.jpg",
    "catalog/kia-sportage-2023/3.jpg"
  ]
}
```

## Ярлыки (badge)

| Текст           | badgeColor      |
|-----------------|-----------------|
| `"Хит продаж"`  | `null` (золото) |
| `"Новое"`       | `"#e8c96a"`     |
| `"Эксклюзив"`   | `null` (золото) |
| `"Trade-in"`    | `"#b0b0b0"`     |
| Без ярлыка      | `badge: null`   |

## Открытие сайта

Сайт работает как обычный HTML-файл. Открыть в браузере:
- Двойной клик на `index.html`, **ИЛИ**
- Через Live Server в VS Code (рекомендуется для работы с локальными фото)
